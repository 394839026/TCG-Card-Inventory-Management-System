TCG 卡牌库存管理系统
简介
纯前端离线版 TCG 卡牌库存管理工具，基于 HTML + Bootstrap5 + JavaScript + LocalStorage 开发，无后端、免安装、开箱即用，浏览器直接打开就能用。
针对大批量卡牌数据做了防抖 + 分页 + 数据缓存深度性能优化，数千上万条卡牌也能秒筛选、秒切换视图，全程无卡顿。
✨ 功能特性
📦 卡牌基础管理
完整字段：编号、名称、版本、稀有度、库存、市场价、成本价、归属、标签、备注
支持单条新增、编辑、删除卡牌
本地 LocalStorage 自动持久化，刷新页面数据不丢失
🖼 多视图 + 分页浏览
三种展示模式：简约列表 / 卡片网格 / 紧凑表格
高性能分页，可自定义每页条数
实时数据统计：总种类、总张数、库存总市值、零库存卡牌数
🔍 多维度智能筛选
关键词模糊检索（编号 / 名称）
按版本、稀有度、归属、自定义标签筛选
按库存状态筛选：有库存 / 零库存
搜索防抖，避免高频输入造成页面卡顿
📥📤 出入库管理
支持入库 / 出库两种操作
输入自动联想匹配卡牌
出库自动校验库存，防止超量出库
可动态新增多行出入库记录，批量提交
📊 导入导出 & 数据备份
下载标准 Excel 模板
增量导入 / 覆盖导入 两种模式
一键导出全量数据为 Excel
JSON 完整备份 / 恢复备份
清空全部数据（二次确认防误删）
⚡ 性能优化
筛选输入防抖，减少无效重渲染
下拉筛选选项缓存，避免重复遍历
分页按需渲染 DOM，不一次性加载海量节点
批量 HTML 拼接，减少页面回流重绘
🚀 快速开始
运行方式
下载项目源码，保存 TCG卡牌管理.html 到本地
使用任意现代浏览器（Chrome / Edge / 360 浏览器等）直接打开
无需部署、无需联网、无后端依赖，纯本地离线运行
初次使用建议
点击「下载模板」获取 Excel 模板
按模板格式批量填写卡牌信息
使用「增量导入」一键导入数据
日常定期导出 Excel 或备份 JSON 防止数据丢失

🛡 数据安全说明
所有数据仅存储在浏览器 LocalStorage，仅本机可见
建议定期手动备份 JSON 文件，防止清理浏览器缓存丢失数据
无痕模式、清除网站数据会清空本地库存，请提前备份
